# 🤖 超星学习通全能答题助手 (AI版)

这是一个基于 Tampermonkey（油猴）的浏览器辅助脚本，旨在帮助用户自动化完成超星学习通（Chaoxing/Mooc）的**作业**、**章节测验**以及**考试整卷预览**。

该脚本通过提取页面题目，调用大模型 API（如 DeepSeek、ChatGPT）获取答案，并自动模拟人工操作填入答案。

## ✨ 核心功能

* **🧠 AI 驱动**：支持自定义 API，可连接 DeepSeek、Kimi、ChatGPT 等任何兼容 OpenAI 格式的接口。
* **💪 强力填空**：内置独家的“穿透写入”技术，完美解决超星 UEditor 编辑器**禁止粘贴**、**无法输入**的问题。
* **🖱️ 全自动操作**：自动识别题目 -> 请求答案 -> 模拟点击/写入 -> 自动翻页/滚动 -> 自动保存。
* **🔇 静默运行**：去除结束时的浏览器弹窗，仅在面板日志中给出完成提示，保持静默运行。
* **💻 可视化面板**：自带悬浮配置窗口，可随时修改 API Key，查看实时进度。

## 🚀 安装指南

1.  **安装浏览器扩展**：
    * 推荐使用 Chrome、Edge 或 Firefox 浏览器。
    * 安装 [Tampermonkey (油猴)](https://www.tampermonkey.net/) 插件。

2.  **安装脚本**：
    * 点击 Tampermonkey 图标 -> "添加新脚本"。
    * 直接将txt文件拖入，等待之后会提示安装。
    * 安装后返回原页面刷新即可

## ⚙️ 配置说明

首次运行脚本时，页面右上角会出现一个**悬浮窗口**。你需要在里面填入以下信息（会自动保存）：

1.  **接口地址 (Base URL)**：
    * DeepSeek (推荐): `https://api.deepseek.com/chat/completions` (默认)
    * 如果你有其他中转站地址，填入对应的 `/v1/chat/completions` 路径即可。
2.  **API Key**：
    * 填入你的 API 密钥（例如 `sk-xxxxxxxx`）。
3.  **模型名称 (Model)**：
    * DeepSeek: `deepseek-chat` (默认)
    * 其他: `gpt-3.5-turbo`, `moonshot-v1-8k` 等。

## 🎮 使用方法

1.  **打开作业/考试页面**：
    * 进入具体的做题页面（支持作业详情页、考试整卷预览页）。
2.  **检查面板**：
    * 确认右上角的“🤖 答题助手”面板已显示。
    * 如果是第一次使用，请先填写配置。
3.  **开始运行**：
    * 点击绿色的 **【开始运行】** 按钮。
    * 脚本会自动滚动到第一题，开始作答。
    * 你可以通过下方的日志窗口观察实时进度。
    * **静默容错**：脚本在后台尝试写入，失败会立即切换方案，最终只在日志中汇报结果。
4.  **结束与检查**：
    * 全部题目完成后，日志会显示 `🎉 全部任务已完成`。
    * **⚠️ 重要**：提交前请务必**人工检查**一遍，特别是填空题和简答题的内容是否通顺、正确。

## ❓ 常见问题

**Q: 填空题没填进去怎么办？**
A: 脚本会尝试所有写入方案，但如果页面结构或编辑器加载特殊，可能仍有极少数失败。日志会显示 `❌ 写入失败`。此时请手动填入。

**Q: 为什么脚本面板不显示？**
A: 请检查：
    1. 脚本是否在 Tampermonkey 中**启用**。
    2. 浏览器地址栏的 **URL 是否匹配**。确保你的 URL 包含 `mooc-ans` 或 `exam-ans` 路径。

**Q: 使用该脚本会被后台检测吗？**
A: 脚本采用了**模拟点击**和**随机延迟**策略，尽量模仿人类操作行为。但任何自动化工具都有风险，请**谨慎使用**，后果自负。
